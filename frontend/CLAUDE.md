# Frontend Assistant Instructions (Next.js)

This guides work on the Next.js frontend located in `/frontend`.

## Context
- **Framework**: Next.js 16+ (App Router)
- **Styling**: Tailwind CSS
- **Port**: 3000

## Core Mandates
1. Follow `../Constitution.md` and `../specs/`.
2. Use `npm` for package management.
3. Prioritize clean, accessible UI components.

## Common Commands
- **Run Dev Server**: `npm run dev`
- **Build**: `npm run build`
- **Lint**: `npm run lint`
