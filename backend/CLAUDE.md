# Backend Assistant Instructions (FastAPI)

This guides work on the Python backend located in `/backend`.

## Context
- **Framework**: FastAPI
- **Toolchain**: `uv`
- **Port**: 8000

## Core Mandates
1. Follow `../Constitution.md` and `../specs/`.
2. Use `uv run` for all commands.
3. Maintain type hints and comprehensive docstrings.

## Common Commands
- **Run Dev Server**: `uv run fastapi dev src/todo_app/main.py`
- **Run Tests**: `uv run pytest`
- **Sync Deps**: `uv sync`
